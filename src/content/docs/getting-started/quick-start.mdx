<Aside type="note" title="Prerequisites">
Before getting started, make sure you have the following installed:

- [**Node.js**](https://nodejs.org/): Version >= 24
- [**Bun**](https://bun.com/): Version >= 1.3

[Learn more](/getting-started/quick-start/#node-and-bun-setup)
</Aside>

<Steps>

1. Initialize a new PageZERO project:
   <Tabs>
      <TabItem label="Open-source edition">
      ```sh
      bunx pagezero@latest init
      ```
      </TabItem>
      <TabItem label="Paid edition">
      ```sh
      bunx pagezero@latest init --powerup
      ```
        <LinkButton href="https://pagezero.dev/#pricing" variant="secondary" icon="external">
          Learn about PowerUP
        </LinkButton>
      </TabItem>
   </Tabs>

2. Enter your project directory:
   ```sh
   cd <your-project-name>
   ```

3. Run development server:
   ```sh
   bun dev
   ```

4. Navigate to [`http://localhost:3000`](http://localhost:3000) to see the app in action.
   That's it! 🎉

</Steps>

## Node and Bun setup

If you are in a rush, already have Node.js installed and want to quickly setup Bun:

```sh
npm install -g bun
```

Otherwise, the recommended way in PageZERO is to use [asdf](https://asdf-vm.com/) runtime version manager:

```sh
asdf plugin add nodejs
asdf plugin add bun
cd <your-project-name>
asdf install
```
