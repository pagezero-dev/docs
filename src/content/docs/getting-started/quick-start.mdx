<Aside type="note" title="Prerequisites">
Before getting started, make sure you have the following installed:

- [**Node.js**](https://nodejs.org/): Version 24 or higher
- [**Bun**](https://bun.com/): Version 1.3 or higher

If you're not sure how to install Node.js or Bun, check out the [Node and Bun setup](#node-and-bun-setup) section.
</Aside>

## Node and Bun setup

If you're in a rush, already have Node.js installed, and want to quickly set up Bun:

```sh
npm install -g bun
```

For a more robust setup, the recommendation in PageZERO is to use the [asdf](https://asdf-vm.com/) runtime version manager.

After installing asdf, run the following commands to install Node.js and Bun:

```sh
asdf plugin add nodejs
asdf plugin add bun
cd <your-project-name>
asdf install
```

This will read the `.tool-versions` file from your project root and automatically install the recommended versions of Node.js and Bun.

## Project structure

PageZERO follows an **integrated monorepo** approach. Unlike typical monorepos, 
package linking happens through TypeScript's path resolver (via `tsconfig.json` paths) rather 
than workspaces. This allows for a simpler yet modular setup.
