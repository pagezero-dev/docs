PageZero is an open-source TypeScript starter for full-stack web development on Cloudflare. It provides a modern, opinionated stack that helps you build fast, scalable applications with excellent developer experience.

## Why PageZero?

<CardGrid stagger>
  <Card title="🚀 Modern Stack" icon="rocket">
    Built with Vite, React, React Router v7, TypeScript, TailwindCSS, and
    Cloudflare services for maximum performance and developer experience.
  </Card>
  <Card title="☁️ Cloudflare Native" icon="external">
    Optimized for Cloudflare Pages (hosting) and D1 (database) with built-in
    deployment workflows and edge computing capabilities.
  </Card>
  <Card title="🛠️ Developer Experience" icon="setting">
    Complete tooling setup with ESLint, Prettier, Vitest, Playwright, Storybook,
    and GitHub Actions CI/CD out of the box.
  </Card>
  <Card title="💰 Cost Effective" icon="star">
    Designed to be cheap to maintain with Cloudflare's generous free tiers and
    efficient architecture patterns.
  </Card>
</CardGrid>

## Key Features

- **🔥 Hot Module Replacement** - Lightning-fast development with Vite
- **📱 Full-Stack TypeScript** - End-to-end type safety from frontend to backend
- **🗄️ Database Integration** - Drizzle ORM with Cloudflare D1 SQLite database
- **🧪 Testing Suite** - Unit tests with Vitest and E2E tests with Playwright
- **📚 Component Library** - Storybook integration for component development
- **🚀 CI/CD Pipeline** - Automated testing, building, and deployment
- **🌐 Edge Computing** - Deploy globally with Cloudflare's edge network
- **💸 Cost Efficient** - Built to leverage Cloudflare's generous free tiers
